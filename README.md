# 🚀 SkillPulse – Learning Dashboard & Skill Tracker

## 📌 Overview

**SkillPulse** is a personal learning dashboard designed to help you track and visualize your skill development journey.

It works like a **GitHub contribution graph — but for your learning**.

With SkillPulse, you can:
- Track the skills you're currently learning (Docker, Kubernetes, Terraform, Go, etc.)
- Log daily/weekly study sessions with notes and time spent
- Visualize your progress through dashboards
- Stay consistent and accountable in your learning journey

---

## ✨ Features

- 📚 Skill Tracking – Add and manage skills you’re learning  
- ⏱️ Study Logs – Record time spent and notes for each session  
- 📊 Progress Dashboard – Visual representation of your learning  
- 🐳 Containerized Setup – Easy deployment using Docker  
- ⚙️ CI/CD Integration – Automated pipeline with Azure Pipelines
